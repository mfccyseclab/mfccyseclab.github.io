# How to collaborate

## To collaborate is to contribute to  other people's project

1. Fork a repository
2. Clone the repository
    `git clone <url-repo>`
3. Open in vs code
    `code <repo-name>`
4. Contribute
5. Save and Commit git change
6. Push to remote server
7. Create pull request (pr)
